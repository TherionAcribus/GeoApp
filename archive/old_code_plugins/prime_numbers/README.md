# Plugin Prime Numbers

Ce plugin permet d'encoder et de décoder des messages en utilisant le code de substitution par nombres premiers.

## Principe
Chaque lettre de l'alphabet est remplacée par un nombre premier selon la table suivante :

| Lettre | Nombre premier |
|--------|----------------|
| A      | 2              |
| B      | 3              |
| C      | 5              |
| D      | 7              |
| E      | 11             |
| F      | 13             |
| G      | 17             |
| H      | 19             |
| I      | 23             |
| J      | 29             |
| K      | 31             |
| L      | 37             |
| M      | 41             |
| N      | 43             |
| O      | 47             |
| P      | 53             |
| Q      | 59             |
| R      | 61             |
| S      | 67             |
| T      | 71             |
| U      | 73             |
| V      | 79             |
| W      | 83             |
| X      | 89             |
| Y      | 97             |
| Z      | 101            |

## Exemples

- **Encodage** :
  - Texte : `DCODE`
  - Résultat : `7,5,47,7,11`

- **Décodage** :
  - Nombres : `53,61,23,41,11`
  - Résultat : `PRIME`
